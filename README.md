# Tower of Hanoi

```
         Rod A                  Rod B                  Rod C
           |                      |                      |
          #|#                     |                      |
         ##|##                    |                      |
        ###|###                   |                      |
       ####|####                  |                      |
      #####|#####                 |                      |
     ######|######                |                      |
    #######|#######               |                      |
   ########|########              |                      |
  #########|#########             |                      |
 ##########|##########            |                      |
---------------------------------------------------------------------
```

`towerofhanoi.py` shows steps to move disks from rod A to rod C in ASCII art animation.

## Usage
```bash
python ./towerofhanoi.py -n 20
```
